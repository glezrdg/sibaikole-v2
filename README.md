# Sibaikole — v2

Rediseño del sitio del estudio de arquitectura y construcción Sibaikole.

## Stack

- Next.js 15 · React 19
- Tailwind CSS 3
- Framer Motion (animaciones)
- Lenis (smooth scroll)
- Fuentes: Fraunces (display) + Inter (cuerpo)

## Desarrollo

```bash
npm install --legacy-peer-deps
npm run dev
```

Abrir `http://localhost:3000`.

## Deploy

Conectado a Vercel.
